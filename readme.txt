

Sudoku Validation Program

This program validates a Sudoku solution stored in a file to determine whether it is a valid solution.
 It uses multi-threading to validate rows, columns, and subgrids concurrently, ensuring efficient validation of the Sudoku solution.

To compile the program,use following command:

make

To run the program:

./mssv <solution_file> <delay>

Example usage:

./mssv solution.txt 3
